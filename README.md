#Tic Tac Toe

**A basic Tic Tac Toe game built using HTML/JavaScript/CSS.** 


## Dependencies
* jQuery

##How to get started
1. To make a move, the player will use the mouse single click to mark a space. In this version, there is no provision to undo the move. As soon as the move is done, the game proceeds to the next playerâ€™s chance.
2. At each move, The game should show whose move (Player A or Player B) it is. When the game ends, it should display one of the following three outcomes:

* Winner: Player A
* Winner: Player B
* Draw
